# Electricity-Billing-System 
The Electricity Billing System is a Java-based desktop application designed to automate the process of electricity bill generation and management. Developed using Java Swing and AWT for the user interface and MySQL as the database, this project provides an efficient and user-friendly solution for handling customer details, meter readings, bill calculations, and payments.

**Key Features:
User Authentication:** Secure login system for administrators and customers.
**Customer Management:** Add, update, and delete customer records along with meter details.
**Bill Generation:** Automatic bill calculation based on electricity usage and tariff rates.
**Payment Management:** Customers can view and pay their electricity bills.
**Billing History:** Stores and retrieves previous billing records for reference.
**Graphical Reports:** Displays consumption trends and payment status.
**Responsive GUI:** Interactive user interface built with Java Swing and AWT for seamless navigation.
**Database Management:** MySQL is used to store customer details, meter readings, bills, and payments securely.

This project demonstrates Core Java concepts, Object-Oriented Programming (OOP), GUI development, and database connectivity using JDBC, making it a valuable addition to any resume for Java development roles.

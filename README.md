The Electricity Billing System is a Java-based desktop application designed to manage electricity billing operations efficiently. It automates the process of generating bills, managing customer details, and processing payments. The system provides an interactive GUI built with Java Swing and AWT and uses MySQL for secure data storage and retrieval.

The project follows the Model-View-Controller (MVC) architecture, ensuring a structured and scalable design. It enables administrators to add customers, update meter readings, and generate bills, while customers can view their bills, check payment status, and make payments.

**Key Features:**
✅ **User Authentication:** Secure login system for administrators and customers.
✅ **Customer Management:** Add, update, and delete customer details.
✅ **Meter Readings & Tariffs:** Automatic bill calculation based on consumption.
✅ **Bill Generation:** Generates detailed electricity bills with due dates.
✅ **Payment Processing:** Tracks bill payments and updates payment status.
✅ **Billing History:** Stores previous bills for future reference.
✅ **Graphical Reports:** Displays consumption trends and billing history.
✅ **Database Integration:** Uses JDBC to connect and manage data in MySQL.
✅ **User-Friendly GUI:** Developed using Java Swing and AWT for smooth navigation.

**Technologies Used:**
🔹 **Java (Core Java) –** Application logic and backend processing.
🔹 **Java Swing & AWT –** User interface development.
🔹 **MySQL –** Database for storing customer and billing details.
🔹 **JDBC –** Database connectivity between Java and MySQL.
🔹 **MVC Architecture –** For modular and structured application design.
🔹 **File Handling (Java I/O) –** For generating reports and logs.
